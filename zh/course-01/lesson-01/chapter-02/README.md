# 第二章: sCrypt 合约

sCrypt 是比特币智能合约的高级开发语言。

一份sCrypt合约就是比特币dApp的基本模块。 所有的变量和函数都属于一份合约。 它是你所有dApp的起点。

sCrypt 合约由关键字 `contract` 定义，类似面向对象语言 Java/C++ 里面的 `class`。一份名为`HelloWorld`的空合约如下:

```

contract HelloWorld {

}

```
